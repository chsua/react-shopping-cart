1. 뭘하려는 걸까

   - 물품 목록이 보여지는 장바구니 페이지 보여주기
   - 물품의 수량을 올리거나 내리면 장바구니에 넣고 빼는것

2. 적당한 난이도로 쪼개기
   - 헤더
     - 장바구니 수량 아이콘
   - 목록
     - 아이템
       - 아이템 정보
       - 수량 (증감버튼)
3. 핵심 - 쉽게 - 적절

   - 수량 + 증감버튼 컴포넌트
   - 헤더 장바구니 수량 컴포넌트

   - 물품 컴포넌트
   - 헤더 컴포넌트
   - 리스트 컴포넌트

4. 실제 상황 예측

   - 수량 + 증감버튼 컴포넌트
